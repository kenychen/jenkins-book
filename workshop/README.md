## Hello! Jenkins!

難度：
簡易

目標：
認識 Jenkins Job 基礎操作

專案：
https://github.com/agileworks-tw/java-hello-world

練習：

1. 新增作業「HelloWorld」
2. 使用 Shell 指令檢查 JDK 版本
3. 使用 Git SCM
4. 使用 Git Parameter 參數化建置切換分支：`BRANCH_NAME`，並帶入 Git SCM 設定
5. 自訂字串參數：`WHO`，並帶入 Shell 指令
